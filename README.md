# Subway-Surfers
ste código cria um aplicativo WebGL para Subway Surfers que carrega os arquivos do jogo a partir de uma CDN e inicia o jogo no navegador. Ele armazena configurações do jogo, como título, versão do Unity e tamanhos dos arquivos, e então combina os arquivos divididos do jogo em arquivos únicos usando a função ` combineFiles`.
