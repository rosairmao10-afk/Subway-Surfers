# Jogo Escape Road car
Este código cria um jogo de corrida infinita 2D simples usando o elemento canvas do HTML. O Jogo jogador pode pular para evitar obstáculos e coletar power-ups que concedem efeitos temporários como escudo, salto alto, câmera lenta ou multiplicador de pontuação. A  gerencia tudo — física (gravidade/salto), geração de obstáculos e power-ups, detecção de colisão, pontuação, animação de fundo e progressão de dificuldade. Quando o jogador atinge um obstáculo sem escudo, o jogo termina, exibindo "Game Over", e a pontuação é comparada com a pontuação máxima salva. Pressionar a barra de espaço faz o jogador pular ou reinicia o jogo após perder.
